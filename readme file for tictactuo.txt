# 🎮 Tic-Tac-Toe AI with Minimax Algorithm

This is a simple command-line Tic-Tac-Toe game where a human player competes against an **unbeatable AI** opponent powered by the **Minimax algorithm with Alpha-Beta Pruning**.

---

## 🧠 Features

- Play against an AI that never loses
- Human vs AI turn-based gameplay
- Uses the Minimax algorithm with Alpha-Beta Pruning for optimal decision-making
- Clean and simple command-line interface
- Configurable starting player (`X` or `O`)

🚀 Getting Started
📋 Prerequisites

Python 3.x

▶️ Running the Game

Clone this repository or copy the script

Save the code as tic_tac_toe_ai.py

Run the game:

python tic_tac_toe_ai.py

📂 File Structure
tic_tac_toe_ai.py      # Main game script
README.md              # Project documentation

🧮 How the AI Works

The AI uses the Minimax algorithm to simulate all possible future moves and picks the optimal one. It also incorporates Alpha-Beta pruning to improve performance by avoiding unnecessary calculations.

Evaluation Function:

+1 if AI wins

-1 if human wins

0 for a draw
